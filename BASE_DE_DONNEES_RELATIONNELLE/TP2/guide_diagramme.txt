═══════════════════════════════════════════════════════════════════════════════════
  RELATIONS (flèches à tracer dans draw.io)
═══════════════════════════════════════════════════════════════════════════════════

  Heros ──── FK id_classe ────────────────────────────────> Classes
       (plusieurs héros peuvent avoir la même classe)
       Côté Heros : N   |   Côté Classes : 1


  Classes <──────────────────── Classes_Competences ───────────────────> Competences
  (1 classe peut avoir plusieurs compétences, 1 compétence peut être dans plusieurs classes)
       Côté Classes : 1   |   Côté Classes_Competences : N
       Côté Competences : 1   |   Côté Classes_Competences : N


  Quetes <──────────────────── Quetes_Monstres ───────────────────────> Monstres
  (1 quête peut avoir plusieurs monstres, 1 monstre peut être dans plusieurs quêtes)
       Côté Quetes : 1   |   Côté Quetes_Monstres : N
       Côté Monstres : 1   |   Côté Quetes_Monstres : N


  Monstres <─────────────────── Monstres_Loots ───────────────────────> Loots
  (1 monstre peut donner plusieurs loots, 1 loot peut venir de plusieurs monstres)
       Côté Monstres : 1   |   Côté Monstres_Loots : N
       Côté Loots : 1   |   Côté Monstres_Loots : N


  Heros ──── FK id_heros ──────────────────────────────────> ListeNoireTaverne
  (1 héros peut être banni plusieurs fois)
       Côté Heros : 1   |   Côté ListeNoireTaverne : N


  Configurations : table ISOLÉE, aucune relation
